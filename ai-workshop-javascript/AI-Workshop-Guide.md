# AI for IT Students and Professionals - Workshop Guide

## 🎯 Workshop Overview

- **Duration:** 45 minutes
- **Level:** Beginner
- **Format:** Interactive hands-on workshop
- **Tools:** Web browser, JavaScript console

## 📋 Learning Objectives

By the end of this workshop, participants will:

1. Understand basic AI and Machine Learning concepts
2. Implement K-Nearest Neighbors classification
3. Build a simple Decision Tree
4. Create a Linear Regression model
5. Apply these concepts using JavaScript

## 🗓️ Workshop Schedule

### Part 1: Introduction to AI (10 minutes)

**Concepts Covered:**

- What is Artificial Intelligence?
- Machine Learning vs Traditional Programming
- 2 types of supervised ML: Classification and Regression
- Real-world applications

**Activities:**

- Interactive discussion
- Review workshop objectives
- Open the workshop interface (`ai-workshop.html`)

### Part 2: Classification Demo (15 minutes)

**Concepts Covered:**

- K-Nearest Neighbors algorithm
- Decision Trees
- Feature vectors and labels
- Iris flower classification

**Hands-on Activities:**

1. **KNN Demonstration:**
   - Input flower measurements
   - Observe predictions
   - Understand distance calculations

2. **Decision Tree Comparison:**
   - Same input, different algorithm
   - Understand rule-based decisions
   - Compare with KNN results

3. **Exercise 1:**
   - Test provided sample measurements
   - Try edge cases
   - Observe prediction differences

### Part 3: Regression Demo (15 minutes)

**Concepts Covered:**

- Linear Regression
- Slope and intercept
- Prediction of continuous values
- House price prediction

**Hands-on Activities:**

1. **Linear Regression Demo:**
   - Input house sizes
   - Observe price predictions
   - View regression equation

2. **Exercise 2:**
   - Test different house sizes
   - Understand the linear relationship
   - Analyze model parameters

### Part 4: Advanced Exploration (5 minutes)

**Concepts Covered:**

- Browser console usage
- Model experimentation
- Adding training data
- Creating custom models

**Hands-on Activities:**

1. **Console Commands:**
   - Open browser developer tools
   - Execute provided code snippets
   - Modify model parameters
   - Add new training data

## 🎓 Instructor Notes

### Presentation Tips

- Start with real-world AI examples (recommendation systems, image recognition)
- Emphasize that AI is about pattern recognition from data
- Use analogies (KNN = asking neighbors, Decision Tree = flowchart)
- Encourage experimentation and questions

### Common Questions & Answers

**Q: "Why JavaScript for AI?"**
A: JavaScript makes AI accessible - runs in any browser, no installation needed, and there are growing libraries like TensorFlow.js and Brain.js.

**Q: "How accurate are these simple models?"**
A: These are educational examples. Real-world models use more data, features, and sophisticated algorithms, but the fundamental concepts are the same.

**Q: "What's the difference between KNN and Decision Trees?"**
A: KNN finds similar examples and uses their labels. Decision Trees create explicit rules. KNN is instance-based, Decision Trees are rule-based.

### Extension Activities

If time permits or for advanced participants:

1. Modify decision tree thresholds
2. Add more training data to KNN
3. Create a new regression dataset
4. Discuss overfitting and validation

### Common Issues

1. **Console errors:** Ensure all files are in same directory
2. **Module import errors:** Use a local web server if needed
3. **Predictions not working:** Check input values are numbers
